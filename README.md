# U8G2_Simulator

U8G2_Simulator is a PC simulator for Arduino platform.You can write UI related code and view the effect on PC.

This fork from: https://github.com/MS10755/U8G2_Simulator.git

Adapted project for Makefile builds.

## Building

### MacOS

+ install SDL:
```bash
brew install sdl2
brew install sdl2_ttf
```
+ build:
```bash
make
```

### Linux Ubuntu:

+ install SDL:
```bash
sudo apt update
sudo apt install libsdl2-dev libsdl2-ttf
```
+ build:
```bash
make
```
